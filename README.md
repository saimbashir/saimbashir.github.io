# Saim Bashir GitHub Pages Site

Static portfolio site for Saim Bashir.

## Profile photo

The site uses:

```text
assets/saim_picture.png
```

Replace that file to update the portrait.

## Publish on GitHub Pages

Create a repository named after the GitHub account, for example:

```text
username.github.io
```

Upload these files to the repository root, then enable GitHub Pages from the repository settings if it is not already active.
